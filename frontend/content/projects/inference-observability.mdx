# Movie Recommendation System with TMDB Integration

## Overview

A Python-based machine learning application that recommends movies based on user preferences using content-based filtering and data from TMDB.

## Project Structure

- **app.py**: The main web application file used to run the user interface.
- **movie_recommendation_system.ipynb**: The notebook used for data preprocessing, exploration, and model building.
- **movies_dict.pkl**: A serialized pickle file containing the processed movie dataset dictionary.
- **similarity.pkl**: A serialized pickle file containing the cosine similarity matrix used for recommendations.
- **requirements.txt**: The list of Python libraries required to run the project.
- **tmdb_5000_movies.csv** and **tmdb_5000_credits.csv**: The raw datasets containing movie details, cast, crew, and metadata.

## Key Features

- Content-based movie recommendations
- TMDB data integration
- Similarity-based matching for personalized suggestions
- Clean Python workflow for model preparation and deployment

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Pickle
- Streamlit or Flask-based UI

## Purpose

This project demonstrates how recommendation systems can be built from real-world movie metadata and presented through a simple interactive application.
