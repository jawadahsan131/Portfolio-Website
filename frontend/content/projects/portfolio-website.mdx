# Portfolio Website

## 🌐 Portfolio Overview

* 🧩 **Modern personal website** — A clean and professional portfolio designed to showcase AI/ML work and software projects.
* 📱 **Responsive design** — Works smoothly across desktop, tablet, and mobile screens.
* 📝 **MDX-based content** — Blog posts and projects are managed through structured Markdown files.
* ⚡ **Fast and SEO-friendly** — Built with Next.js and optimized for performance and search visibility.

---

## ✨ Key Features

* 🖼️ **Gallery section** — Displays visual highlights and project aesthetics.
* 📂 **Projects page** — Shows completed work with descriptions, links, and tags.
* 📰 **Blog section** — Supports published articles and technical writing.
* 📄 **Resume page** — Presents education, experience, and skills in a polished layout.
* 📬 **Contact form** — Sends messages through a FastAPI backend for email delivery.

---

## 🛠️ Tech Stack

* **Frontend:** Next.js, React, Tailwind CSS
* **Content:** MDX, Gray Matter
* **Backend:** FastAPI
* **Deployment:** Vercel + Render

---

## 🎯 Purpose

This website was created to present my work professionally while keeping the content easy to update and scale over time.
